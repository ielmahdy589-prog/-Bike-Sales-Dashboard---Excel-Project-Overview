# 🚴 Bike Sales Dashboard - Excel Project

## 📌 Overview

This project analyzes bike sales data to understand customer behavior, identify trends, and generate actionable business insights using Excel.

---

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Data Visualization

---

## 📊 Key Insights

* Higher income customers are more likely to purchase bikes
* The 31–54 age group is the dominant customer segment
* Short commute distances are strongly associated with higher purchase rates

---

## 💡 Recommendations

* Focus on high-income customer segments
* Target the 31–54 age group in marketing campaigns
* Promote bikes as a short-distance commuting solution

---

## 📁 Project Files

* Raw Dataset
* Cleaned Data
* Pivot Tables
* Final Dashboard

To demonstrate data analysis and dashboarding skills using Excel.
